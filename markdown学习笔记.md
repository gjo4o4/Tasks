# markdown学习笔记

## markdown标题

- ### 使用 = 和 - 标记一级和二级标题

  eg：

```
一级标题
=======
二级标题
-------
```

- ### 使用#号标记

  使用 **#** 号可表示 1-6 级标题，一级标题对应一个 **#** 号，二级标题对应两个 **#** 号，以此类推。

  eg：

  ```
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
  ###### 六级标题
  ```

- ### 注意事项：

  1. **# **号和标题文字之间需要有一个空格

  eg：

  ```
  # 正确
  #错误
  ```

  

  2. **#**号必须在行首，前面不能有任何的字符。

  3. 一个文档通常只使用一个一级标题作为文档的主标题。

  4. 一般不超过 10 个汉字或 20 个英文字符。

  5. 使用描述性词语，避免模糊的标题如"其他"、"杂项"。

- ### 嵌套结构

  标题的层次结构应该遵循逻辑顺序，不应该跳级使用。

  eg：

  ```
  推荐：
  # 主题
  ## 二级标题
  ### 三级标题
  避免：
  # 主题
  ### 三级标题
  ## 二级标题
  ```
